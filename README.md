# Fitness Tracker App

## Description
A Fitness Tracker App built using Python, Streamlit and SQLite.

## Features
- Track daily steps
- Track workout time
- Track calories burned
- Select exercise type
- Save fitness records
- Dashboard with charts

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
